The One Repository for The One Tool mod for Minecraft.

Description:
* Adds a tool whose purpose is to function like all vanilla tools in one and more.

Built-in vanilla functions:
* Pickaxe
* Axe (+ stripping)
* Shovel (+ pathing)
* Hoe (+ tilling)
(For the last two, it will path grass-type blocks but till dirt-type blocks)

Extra functions:
* Edible. Fills hunger and gives half as many as it filled as Saturation. (ex. if you eat empty it will give full hunger + half saturation)

Planned function(s):
* Place torches on Stone from Player Inventory.

The code in this repository is probably terrible, so viewer discretion is advised.
